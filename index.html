What this repository is

This GitHub repository is an evidence archive and review surface. It should not be treated as the product interface for live biomedical computation.

The intended use is to review already executed reports, verify SHA-256 receipts, inspect Tier-0 boundaries, and understand why each claim was killed, retained as data-only, or left on watchlist.

Strict boundary

This release is not a medical device, diagnostic tool, treatment recommendation system, clinical decision-support product, or wet-lab validation.

APM/B2M–TAP–HLA is retained only as a data-follow-up axis. Killed claims remain killed as written under current Tier-0 evidence.
